# MyMessageCenter
MyMessageCenter 开源个人消息中心，用于接收管理同步个人的消息
1. 主要定义功能、交互标准、交互接口
2. 前后端分离
3. 提供简单实现用于用户自搭建
4. 提供一个在线的站点
